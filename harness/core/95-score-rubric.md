# Harness Score Rubric

100점 하네스 기준.

| 항목 | 배점 |
|---|---:|
| 거짓말 금지와 팩트 기반 보고 | 15 |
| 최소 코드와 불필요 코드 삭제 | 15 |
| 코드 품질과 코드/디자인 통일성 | 15 |
| 토큰 절약 라우팅 | 10 |
| 백업, git 안전, 복구 가능성 | 10 |
| 보안, 사업성, UI/UX 점검 | 15 |
| CAD LSP/인코딩/도면 안전 | 10 |
| 팀 프로젝트 적용성 | 10 |
| skill 설치와 새 프로젝트 적용 자동화 | 5 |
| 검증 스크립트, 작업 카드, 운영 가드레일 | 5 |

## 점수 해석
- 95-100: 실전 기본값으로 사용 가능
- 85-94: 강하지만 적용 자동화나 토큰 효율 개선 필요
- 70-84: 규칙은 있으나 운영 장치 부족
- 69 이하: 프로젝트마다 흔들릴 가능성 큼

자동 점검은 아래 명령으로 실행한다.

```powershell
powershell -ExecutionPolicy Bypass -File scripts/audit-harness.ps1 -Root .
```
