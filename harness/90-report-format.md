# Report Format

보고는 짧고 실용적으로 한다.

## 구현/수정 완료
형식:

```text
완료했어.
바뀐 점: ...
정리한 코드: ...
검증: ...
불확실한 점: ...
남은 위험: ...
```

## 기획/설계 완료
형식:

```text
정리했어.
방향: ...
MVP: ...
제외할 것: ...
근거/불확실성: ...
다음 작업: ...
```

## 외주 검토 완료
형식:

```text
검토했어.
문제: ...
확인할 질문: ...
검수 기준: ...
팩트/추론/불확실성: ...
수정 요청안: ...
```
