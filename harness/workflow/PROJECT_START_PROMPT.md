# Project Start Prompt

새 프로젝트를 시작할 때 아래 문장을 사용한다.

```text
이 프로젝트는 Codex Work Harness 규칙을 따른다.

먼저 AGENTS.md와 harness/00-start-here.md를 기준으로 작업을 분류해줘.
외주를 맡기는 프로젝트인지, 내가 직접 개발할 프로젝트인지 판단해줘.
Simple/Connected/Scale 중 프로젝트 레벨도 판단해줘.
아이디어 단계라면 MVP와 제외할 기능부터 정리하고,
구현 단계라면 필요한 skill과 관련 파일만 읽어줘.

코드는 반드시 최소한으로 짧게 작성하고,
수정/추가 후 필요 없어진 코드는 삭제해줘.
파일은 유지보수와 재사용이 쉽도록 역할별로 분리하되,
과한 추상화와 미래 대비용 코드는 만들지 마.
작업 후 필요한 검증을 하고 짧게 보고해줘.
```
