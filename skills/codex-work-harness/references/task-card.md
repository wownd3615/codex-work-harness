# Task Card

Use a Task Card when a task is too large for direct editing but too small for full planning documents.

## Use When
- Three or more files may change.
- The work has multiple steps.
- Work may pause and resume later.
- Subagents may be used.
- Deletion, migration, deployment, or data changes create risk.
- Outsourced or team review needs a simple shared record.

Skip it for tiny fixes.

## Location
Default:

```text
docs/00-task-cards/{task-name}.md
```

Use `templates/task-card-lite.md`.

## Required Sections
- Goal
- Context
- Constraints
- Steps
- Progress
- Decisions
- Verification
- Cleanup

Keep it short. If the Task Card slows the work down, it is too heavy.
