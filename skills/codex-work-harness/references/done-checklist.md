# Done Checklist

Before completion, check:
- Request is satisfied.
- Code is short and simple.
- No unnecessary abstraction.
- Obsolete code is removed.
- Unused imports, variables, functions, components, and files are cleaned.
- File separation is useful, not excessive.
- Existing style is followed.
- Tests or practical verification were run when feasible.
- If verification was not possible, report why.
